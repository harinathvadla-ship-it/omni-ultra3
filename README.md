An autonomous, single-file AI agent designed for high-performance mobile deployment. Optimized for Android WebView environments (AppsGeyser) and hardware-level technical troubleshooting.

## 🚀 Overview
OMNI-CORE is a terminal-style AI interface built specifically for the **harinathvadla-ship-it** ecosystem. It bridges the gap between raw AI processing and mobile utility, featuring a custom "Thinking Loop" protocol for more accurate technical responses.

### Key Features
* **Zero-Host Deployment:** Runs entirely as a single HTML/JS file.
* **Thinking Protocol:** Displays its logical steps before providing final output.
* **Hardware Optimized:** High-contrast UI designed for Vivo AMOLED displays to reduce battery drain.
* **Developer Workflow:** Integrated "Copy Output" function for rapid code and command transfer.

## 📱 Mobile Installation (AppsGeyser)
1. Copy the source code from `index.html`.
2. Go to **AppsGeyser** and select the **HTML Template**.
3. Paste the code and name the app **OMNI-CORE**.
4. Generate the APK and install it on your device.

## 🛠 Technical Details
* **Frontend:** Vanilla HTML5 / CSS3 / JavaScript.
* **AI Engine:** Google Gemini 1.5 Flash API.
* **Security:** Base64 credential masking for repository safety.
